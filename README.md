# Installation
1. Clonez le dépôt : `git clone https://github.com/yourusername/appliquizz.git`
2. Accédez au répertoire du projet : `cd tp1`
3. Installez les dépendances : `flutter pub get`
4. Lancez l'application : `flutter run`
# Configuration
Changez l'URL de l'API dans `lib/app_config.dart` pour pointer vers votre serveur.
# Fonctionnalités
- Inscription : Les utilisateurs peuvent créer un compte en fournissant un nom d'utilisateur, une
adresse e-mail et un mot de passe.
- Connexion : Les utilisateurs peuvent se connecter à l'application en utilisant leur adresse e-mail et
leur mot de passe.
- Tableau de classement : Les utilisateurs peuvent voir un tableau de classement affichant les meilleurs
scores des joueurs.
# Technologies utilisées
- Flutter : Framework de développement d'applications mobiles.
- Dart : Langage de programmation utilisé pour développer l'application.
- HTTP : Package pour effectuer des requêtes HTTP.
- Shared Preferences : Package pour stocker des données localement sur l'appareil.
